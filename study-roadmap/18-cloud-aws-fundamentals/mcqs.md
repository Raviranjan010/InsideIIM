# Cloud Practice: 12 Conceptual MCQs

These questions test your understanding of cloud definitions, core AWS service functions, and architectural decisions.

---

### Q1. What does the term "serverless" mean in cloud computing?
- [ ] A) The application runs without any servers.
- [ ] B) The developer does not need to manage, configure, or patch the underlying server operating system; they only upload and pay for executing code.
- [ ] C) The database does not store text.
- [ ] D) The website only works offline.

**Correct Answer:** B
*   **Why it's correct:** In serverless architectures (like AWS Lambda), servers still run, but the cloud provider handles all management and provisioning tasks.
*   **Why other options are wrong:** A is technically impossible. C and D are incorrect.

---

### Q2. Which AWS service is designed to host virtual machine servers that you configure and control?
- [ ] A) Amazon S3
- [ ] B) Amazon RDS
- [ ] C) Amazon EC2
- [ ] D) AWS Lambda

**Correct Answer:** C
*   **Why it's correct:** EC2 (Elastic Compute Cloud) provides virtual machine instances where you can configure the OS, run background scripts, and install custom servers.
*   **Why other options are wrong:** A is file storage. B is managed databases. D is serverless execution.

---

### Q3. If you want to store 10,000 PDF financial reports generated by your agent, which AWS service is the most cost-effective and appropriate choice?
- [ ] A) Amazon EC2
- [ ] B) Amazon S3
- [ ] C) Amazon RDS
- [ ] D) AWS Lambda

**Correct Answer:** B
*   **Why it's correct:** Amazon S3 (Simple Storage Service) is optimized for storing static files, images, PDFs, and backups at low cost with high durability.
*   **Why other options are wrong:** EC2 would store them on server disks, which is more expensive. RDS is for database records, not file binary uploads. Lambda is for code execution.

### Q4. Under what billing model does AWS Lambda operate?
- [ ] A) Flat monthly server fee.
- [ ] B) Pay per millisecond of code execution time.
- [ ] C) Flat fee per line of code.
- [ ] D) Completely free.

**Correct Answer:** B
*   **Why it's correct:** Lambda charges are based on the number of requests and the duration (milliseconds) it takes for your code to execute.
*   **Why other options are wrong:** A describes traditional VPS hosting. C and D are incorrect.

### Q5. What is the role of Amazon RDS in a cloud architecture?
- [ ] A) To display user interfaces.
- [ ] B) To host and manage relational database engines (like PostgreSQL) with automated backups and scaling.
- [ ] C) To run background JavaScript loops.
- [ ] D) To manage API route keys.

**Correct Answer:** B
*   **Why it's correct:** RDS (Relational Database Service) simplifies database administration by managing backups, patches, and scaling for relational engines.
*   **Why other options are wrong:** A describes frontend frameworks. C describes EC2/containers. D is handled by API gateways.

### Q6. Why is Amazon S3 preferred over local EC2 server disks for storing user-uploaded files?
- [ ] A) S3 is faster than local RAM.
- [ ] B) EC2 disks are ephemeral (can be wiped if the server restarts or scales down), whereas S3 is durable and persistent.
- [ ] C) S3 only stores text.
- [ ] D) EC2 cannot connect to databases.

**Correct Answer:** B
*   **Why it's correct:** EC2 instances can be destroyed or scaled down dynamically. Storing files on EC2 disks risks losing them during updates. S3 persists files independently.
*   **Why other options are wrong:** A is false. C is false. D is incorrect.

### Q7. What does the cloud concept "Elasticity" refer to?
- [ ] A) The ability to encrypt files.
- [ ] B) The capability of cloud resources to automatically scale up or scale down to match changing demand.
- [ ] C) Writing code that supports multiple languages.
- [ ] D) Testing databases.

**Correct Answer:** B
*   **Why it's correct:** Elasticity enables systems to scale down during low traffic to save costs, and scale up during traffic spikes to maintain performance.
*   **Why other options are wrong:** A describes cryptography. C describes internationalization. D is testing.

### Q8. If your AI agent needs to run a 5-minute HTML scraping and text analysis job every hour, which AWS service is the most cost-efficient choice?
- [ ] A) An EC2 instance running continuously.
- [ ] B) AWS Lambda triggered by a scheduled event.
- [ ] C) A local Express server.
- [ ] D) Amazon S3.

**Correct Answer:** B
*   **Why it's correct:** Since the job only runs for 5 minutes per hour, using Lambda means you only pay for those 5 minutes of compute time.
*   **Why other options are wrong:** A is more expensive because you pay for the remaining 55 minutes of idle time. C is a local machine. D is storage.

### Q9. What does the abbreviation "OpEx" mean in cloud finance discussions?
- [ ] A) Operating Expense (paying monthly for services consumed).
- [ ] B) Capital Expense (buying hardware upfront).
- [ ] C) Optional Encryption.
- [ ] D) Optimized Execution.

**Correct Answer:** A
*   **Why it's correct:** Cloud computing shifts expenditures from capital investments (CapEx) to monthly operational expenses (OpEx) based on consumption.
*   **Why other options are wrong:** B describes CapEx. C and D are incorrect terms.

### Q10. In RDS, what is the purpose of a "Multi-AZ" deployment?
- [ ] A) To compile the database to multiple languages.
- [ ] B) To replicate your database in a secondary physical data center (Availability Zone) for automated failover if the primary data center goes offline.
- [ ] C) To speed up SQL search queries.
- [ ] D) To clear cached search logs.

**Correct Answer:** B
*   **Why it's correct:** Multi-AZ (Availability Zone) replication provides high availability. If the primary zone fails, RDS automatically switches to the standby replica.
*   **Why other options are wrong:** A, C, and D are incorrect.

### Q11. Which AWS service is best suited to manage API access tokens, rate limiting, and request routing at the entry point of your system?
- [ ] A) Amazon S3
- [ ] B) Amazon API Gateway
- [ ] C) Amazon RDS
- [ ] D) AWS Lambda

**Correct Answer:** B
*   **Why it's correct:** API Gateway acts as the entry point, managing routing, rate-limiting, and access tokens for backend APIs.
*   **Why other options are wrong:** A is storage. C is database. D is serverless execution.

### Q12. If a developer commits their AWS Access Key ID to a public GitHub repository, what is the primary risk?
- [ ] A) The repository will load slower.
- [ ] B) Automated scanners will steal the key, allowing unauthorized users to run resources (like GPU miners) on their account, incurring massive bills.
- [ ] C) Git will throw a merge conflict.
- [ ] D) Next.js will stop compiling.

**Correct Answer:** B
*   **Why it's correct:** Publicly exposed AWS keys are targeted by automated scrapers to spin up compute resources (like crypto-miners) on your budget.
*   **Why other options are wrong:** A, C, and D are incorrect.
